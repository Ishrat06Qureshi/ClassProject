05. Time Management

Time Management

Think about when you'll be able to fit your Nanodegree program into your schedule. We recommend blocking off at least 4-5 separate learning sessions every week, with each one lasting 2-3 hours.

Please go into your calendar and schedule time to work on your Nanodegree program for the next two weeks. Remember, consistent and frequent learning is the key to achieving your learning goals!

Study Plan for the First 2 Weeks:

Week 1

Time Activity Notes
1 hr Complete the “Welcome to the React Nanodegree Program” lesson.
1.5 hrs Complete lesson 1 (Why React). In this lesson, you'll discover what makes React such a popular JavaScript library.
2 hrs Complete lesson 2 (Rendering UI with React). In this lesson, you'll learn to create React elements and scaffold your projects.
5.5 hrs Complete the following sections of lesson 3: Introduction, Pass Data Into Components With Props, Exercises - Preface, Functional Components, Add State To A Component, Update state with setState, PropTypes, and Controlled Components. Complete the following coding exercises in lesson 3: Exercise 1 - Passing Data, Exercise 2 - Passing Data, Exercise - Functional Components, Exercise - Managing State, Exercise 1 - Controlled Components, and Exercise 2 - Controlled Components. You'll learn how to pass data into a component, how to create Functional Components, when and how to use PropTypes, and how to create controlled components.

Week 2

Time Activity Notes
6 hrs Complete the following sections of lesson 3: Lesson Summary. Complete the following coding exercises in lesson 3: Exercise 1 - All Together and Exercise 2 - All Together. You’ll practice your newly acquired skills by completing 2 mini projects.
1 hr Complete lesson 4 (Lifecycle Events). In this lesson, you'll learn how to fetch data in a React application and use different lifecycle hooks.
2 hrs Complete lesson 5 (React Router). In this lesson, you'll learn how to add routing to a React app by using React Router.
